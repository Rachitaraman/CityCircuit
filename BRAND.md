## üìä Typography

### Primary Font Stack
- **Web**: `system-ui, -apple-system, BlinkMacSystemFont, sans-serif`
- **Fallback**: `Arial, Helvetica, sans-serif`
- **Characteristics**: Clean, modern, highly readable

### Hierarchy
- **H1**: 48px, Bold (700) - Main headlines, hero text
- **H2**: 32px, Bold (700) - Section headers, page titles
- **H3**: 24px, Bold (700) - Subsections, card titles
- **H4**: 20px, SemiBold (600) - Component headers
- **Body**: 16px, Regular (400) - Main content, descriptions
- **Small**: 14px, Regular (400) - Secondary info, captions
- **Micro**: 12px, Medium (500) - Labels, metadata

### Font Weights
- **Light (300)**: Subtle text, large displays
- **Regular (400)**: Body text, descriptions
- **Medium (500)**: Labels, emphasis
- **SemiBold (600)**: Subheadings, buttons
- **Bold (700)**: Headlines, strong emphasis
